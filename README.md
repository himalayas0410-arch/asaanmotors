# Asaan Motors Driving Training School

A modern, premium website for Asaan Motors Driving Training School, offering professional driving training with advanced simulator practice and real-world road experience to build safe and confident drivers.

## Features

- **Advanced Driving Simulator**: High-fidelity simulator training to build confidence before hitting the road.
- **Experienced Instructors**: Personalized training with professional, patient instructors.
- **Dual-Control Vehicles**: Modern, fully-equipped training cars ensuring 100% safety on the road.
- **Structured Courses**: Multiple programs tailored for beginners, intermediates, and working professionals.

## Running Locally

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1. Install the required dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local address (typically `http://localhost:3000`).

## Tech Stack
- React 19
- TypeScript
- Vite
- Tailwind CSS
- Motion (for animations)
- Lucide React (icons)
